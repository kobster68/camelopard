# Camelopard

Camelopard is a Minecraft Forge mod for Minecraft 1.12.2.

## Status

This project is in early development.

## Requirements

- Minecraft 1.12.2
- Minecraft Forge 14.23.5.2864
- Java 8

## Build from Source

From the `camelopard` root folder:

```powershell
.\gradlew build
```

The jar will be built at:

`build/libs/camelopard-#.#.#.jar`
